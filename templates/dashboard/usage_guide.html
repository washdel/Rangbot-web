Panduan Penggunaan RangBot

Pelajari cara menggunakan perangkat RangBot dengan mudah dan efektif

1

Persiapan Awal

Sebelum menggunakan perangkat RangBot, pastikan semua persiapan telah dilakukan dengan baik untuk memastikan operasi yang optimal.

  • Pastikan perangkat RangBot sudah terhubung ke sumber daya listrik dengan stabil
  • Periksa koneksi internet untuk akses kontrol via website dan sinkronisasi data
  • Pastikan semua komponen perangkat dalam kondisi baik dan tidak ada kerusakan
  • Verifikasi bahwa kamera dan sensor berfungsi dengan baik
Waktu persiapan: 5-10 menit
2

Menghidupkan Perangkat

Proses menghidupkan perangkat RangBot sangat sederhana. Ikuti langkah-langkah berikut untuk memastikan perangkat siap digunakan.

  • Tekan dan tahan tombol power pada perangkat selama 2-3 detik
  • Tunggu hingga indikator LED menyala hijau yang menandakan perangkat aktif
  • Periksa status perangkat di dashboard untuk memastikan koneksi berhasil
  • Tunggu hingga semua sistem selesai booting (sekitar 30-60 detik)
Indikator hijau = Perangkat siap digunakan
3

Kontrol via Website

Akses halaman kontrol perangkat di dashboard untuk mengoperasikan RangBot secara remote dari mana saja. Kontrol penuh di ujung jari Anda.

  • Buka menu "Manajemen Rangbot" di dashboard untuk melihat daftar perangkat
  • Pilih perangkat yang ingin dikontrol dan klik untuk masuk ke halaman kontrol
  • Pilih blok lahan yang ingin dikontrol menggunakan peta interaktif
  • Gunakan tombol kontrol untuk menggerakkan perangkat (maju, mundur, kiri, kanan)
  • Monitor pergerakan melalui streaming kamera real-time
Kontrol dari mana saja, kapan saja
4

Deteksi Penyakit Stroberi

Gunakan teknologi AI untuk mendeteksi penyakit tanaman stroberi secara akurat. Dua metode tersedia untuk deteksi yang fleksibel.

  • Streaming Real-time: Aktifkan streaming kamera untuk monitoring langsung dan deteksi otomatis
  • Upload Foto: Upload foto tanaman untuk deteksi manual menggunakan teknologi YOLO
  • Lihat hasil deteksi di riwayat dengan informasi lengkap (jenis penyakit, tingkat keparahan, lokasi)
  • Gunakan data riwayat untuk analisis tren dan perawatan preventif
Teknologi AI untuk deteksi akurat
5

Monitoring Data Sensor

Pantau kondisi lingkungan kebun Anda secara real-time melalui data sensor yang terintegrasi dengan perangkat RangBot.

  • Akses menu "Data Sensor" di halaman manajemen perangkat
  • Lihat data real-time: suhu, kelembaban, pH tanah, dan parameter lainnya
  • Analisis grafik historis untuk melihat tren perubahan kondisi lingkungan
  • Gunakan data untuk optimasi perawatan tanaman dan pencegahan masalah
Data real-time dari Firebase

Tips & Trik

Koneksi Stabil

Pastikan koneksi internet stabil untuk kontrol real-time dan streaming yang lancar.

Perawatan Rutin

Lakukan pengecekan rutin setiap minggu untuk memastikan semua komponen berfungsi optimal.

Kualitas Foto

Untuk hasil deteksi terbaik, gunakan foto dengan pencahayaan yang cukup dan fokus yang jelas.

Review Riwayat

Rutin review riwayat deteksi untuk mengidentifikasi pola dan mengambil tindakan preventif.
