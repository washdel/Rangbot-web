# Image Placeholders

Tempatkan gambar-gambar berikut di folder ini:

1. robot-illustration.png - Ilustrasi robot RangBot
2. logo.png - Logo RangBot
3. hero-bg.jpg - Background untuk hero section
4. strawberry-farm.jpg - Foto kebun stroberi
5. disease-detection.jpg - Ilustrasi deteksi penyakit
6. dashboard-preview.png - Screenshot dashboard

Ukuran yang disarankan:
- Logo: 512x512 px (PNG dengan transparent background)
- Hero illustration: 800x800 px
- Background images: 1920x1080 px
- Feature icons: 128x128 px

Format yang didukung: JPG, PNG, SVG, WebP
